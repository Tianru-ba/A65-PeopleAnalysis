# A65 项目人力成本经营分析看板

这是一个基于 Streamlit 构建的专业、交互式人力成本经营分析仪表盘。旨在整合 A65 项目的历史实际数据与未来预测，提供深度的人力规模、成本结构及预算执行洞察。

## 🚀 核心功能
- **月度成本监控**：实时追踪 MoM 成本变动及环比率。
- **人力规模趋势**：整合实际与预测人数的动态变化。
- **结构深度穿透**：通过旭日图（Sunburst）穿透职能与岗位性质。
- **人均效率分析**：各职能人均月投入成本构成拆解。
- **Burn Rate 预警**：对比历史月均基准线与未来预算消耗。

## 📊 视觉与交互标准
- **UI 设计**：极致大气的大厂经营分析风格，高辨识度字号。
- **交互科技感**：支持“点击提亮、周围变暗”的聚焦模式，配有坐标轴十字准心。
- **数据透明**：所有核心指标均配有计算逻辑悬停说明。

## 🛠️ 如何运行
1. 安装依赖：`pip install -r requirements.txt`
2. 启动应用：`streamlit run app.py`

## 📁 文件说明
- `app.py`: 核心看板应用程序代码。
- `requirements.txt`: 项目所需的 Python 依赖库。
- `A65汇总及明细.xlsx`: 核心数据源（含明细、汇总及预测数据）。
- `A65项目人力成本看板需求规格说明书.md`: 详细的项目需求与 UX 标准文档。
