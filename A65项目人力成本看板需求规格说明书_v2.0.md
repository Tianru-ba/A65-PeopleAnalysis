# A65 项目人力成本经营分析看板 - 需求规格说明书 (PRD)

**版本**：V2.0（UI/UX 体验升级版）  
**项目名称**：A65 人力成本分析仪表盘  
**日期**：2026-05-06  

---

## 1. 项目背景与目标

在 V1.0 核心逻辑基础上，引入互联网大厂级 B 端视觉规范。通过高呼吸感的布局、严谨的字体层级与渐进式信息暴露方式，提升决策层在长周期项目（2025.07 - 2026.12）中的看数效率。

## 2. 数据源与清洗定义

**文件名称**：A65汇总及明细.xlsx  
**核心页签**：明细（Sheet1）、预测明细  

**自动化要求**：
- 系统需具备“同义词自动映射”能力（示例：将“职务”映射为“职能分类”）
- 系统需处理 Excel 内部日期序列号（兼容日期列存在字符串/数字/序列号等混合形式）

## 3. 功能需求与维度设计

### 3.1 导航与筛选 (Interaction & Filtering)

**顶部级联导航**：
- 采用胶囊式标签页（Nav-pills）切换分析视角  
  - 维度一：整体成本  
  - 维度二：职能透视  
  - 维度三：预算执行  

**侧边栏抽屉**：
- 收纳“年份、季度、月份、职能”的多选筛选器
- 保持主界面整洁（侧边栏可折叠），展开后直接展示筛选项（不额外增加“展开筛选条件”二次折叠）
- 时间维度：年度 → 季度 → 月份（级联联动，多选）
- 职能分类：多选

### 3.2 核心指标卡 (Global Metrics)

**四大核心指标**：
- 累计总成本
- 当前总人数
- 本月总支出
- 人均单价

**视觉表现**：
- 指标卡：纯白圆角卡片
- 边框：rgba(0,0,0,0.06)
- 投影：0 12px 30px rgba(0,0,0,0.06)
- 数值：主色强调（品牌蓝）并加粗

### 3.3 可视化核心看板

**月度趋势分析**：
- 采用堆叠面积图/柱状图展示固定薪酬 vs 奖金激励
- 辅助双轴展示人数变动折线

**结构深度穿透**：
- 使用旭日图（Sunburst）或热力图（Heatmap）分析不同月份下，各职能岗位的成本分布密度

**预测 vs. 实际 (Burn Rate)**：
- 展示历史实际数据与未来（2026.04 - 2026.12）预测数据的衔接
- 预测部分使用虚线或填充纹理区分
- 横坐标按月连续显示（示例：2025-07、2025-08、2025-09… 不间隔）

## 4. UI/UX 视觉标准（视觉核心）

### 4.1 配色方案 (Color Palette)

**背景色调**：
- 全局背景：#ffffff（纯白）
- 侧边栏背景：#ffffff（与主页面一致，减少分割感）

**强调色**：
- 主色（科技蓝）：#0d6efd（用于选中状态、主趋势线及关键指标数值）
- 选中阴影：rgba(13,110,253,0.18)（用于活跃标签的悬浮光效）

**卡片样式**：
- 背景：白色
- 描边：rgba(0,0,0,0.06)
- 投影：0 12px 30px rgba(0,0,0,0.06)
- 选中态（Nav-pills）：蓝底白字 + 轻阴影（不使用底部红色指示线）

### 4.2 字体规范 (Typography)

**层级逻辑**：
- 全局基准（Body / Labels / Table）：20px
- 主标题（App-title）：36px，700 权重，字间距 0.2px
- 模块标题（Section-title）：30px，800 权重
- 卡片标题（Panel-title）：22px，800 权重（加粗）
- 元信息行（Meta-line）：20px，颜色 #6c757d
- 指标卡（Metric）
  - 标题：24px，900 权重
  - 数值：42px，800 权重（品牌蓝强调）
- 导航胶囊（Nav-pills Tab）：20px，800 权重（选中态白字+蓝底+轻阴影）
- 看数指南（Guide）：16px，500 权重（浅色底框，文字次级灰，视觉层级低于小标题）
- 图表（Plotly）
  - 图表标题：22px
  - 坐标轴/图例：18px
  - 提示框（Hover）：19px

### 4.3 元数据处理 (Meta Data)

**副标题弱化**：
- 在主标题下方使用灰色（#6c757d）与 20px 字号标注数据源、统计周期、口径等信息，降低干扰提升效率

**看数指南**：
- 在每个图表下方增加极简文字说明，解释指标背后的经营含义
- 呈现方式：浅色底框区分区域，但避免叠加花哨效果，确保清晰明了

### 4.4 交互体验 (UX)

**玻璃拟态导航栏**：
- 顶部 Header：rgba(255,255,255,0.95)
- backdrop-filter: blur(8px)
- 滚动时呈现半透明毛玻璃效果

**筛选器可读性**：
- 下拉/多选控件：白底 + 轻描边（rgba(0,0,0,0.06)）
- 聚焦态：蓝色描边 + 轻外发光（rgba(13,110,253,0.18)）
- 侧边栏字体：基准 16px（标题/标签/下拉输入按 16px 体系同步缩小）
- 已选项标签（Tag/Chip）：浅蓝底（rgba(13,110,253,0.08)）+ 胶囊形，描边 rgba(13,110,253,0.18)，文字品牌蓝（Tag 字号 14px）
- 选中/悬停项：浅蓝高亮（选中 rgba(13,110,253,0.10) / 悬停 rgba(13,110,253,0.06)，不使用红色体系）

**渐进式查看**：
- 使用折叠面板（Accordion/Expander）收纳明细台账
- 默认隐藏，仅在需要核对数据时点开，避免“表格墙”

**响应式布局**：
- 使用流式容器布局，在不同分辨率下自动调整图表比例
- 确保移动端（如飞书）可正常阅读与交互

## 5. 技术实现建议

**样式注入**：
- 在 Streamlit 中通过 st.markdown 注入自定义 CSS 类
- 实现卡片圆角、投影、胶囊标签、弱化元信息等视觉规范

**动态交互**：
- 使用 st.tabs 实现导航切换，并通过 CSS 映射为 Nav-pills 视觉
- 保持交互逻辑一致性与可维护性
